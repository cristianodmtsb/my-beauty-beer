## My Beauty Beer

Neste projeto de teste, foram usadas técnicas de:

- ReactJs
- React Redux
- Redux Saga
- Styled Component
- Components
- Reactotron

Foi usada a api:
PUNK API
[PUNK API](https://punkapi.com/)

### Rodando

Para instalar use:

`yarn install`

Ou

`npm install`

Para startar o server use:

`yarn start`

ou

`npm start`

Acesse: [http://localhost:3000](http://localhost:3000)
